# Modules Assignment

This package demonstrates simple utility modules for mathematics, string manipulation, and a shop package with billing and discount functionality.

## Structure

- `math_utils.py` - Basic math functions (`add`, `subtract`, `square`).
- `string_utils.py` - String utility functions (`capitalize_words`, `reverse_string`, `word_count`).
- `shop_package/` - Package containing:
  - `billing.py` (`calculate_total`, `apply_tax`)
  - `discount.py` (`apply_discount`, `flat_discount`)
  - `__init__.py` exports core symbols.
- `main.py` - Example script demonstrating imports and usage.

## How to Run

1. Ensure you have Python 3.6+ installed on your system.
2. Open a terminal and navigate to the `modules_assignment` directory:
   ```bash
   cd d:\code\ML\pyth\modules_assignment
   ```
3. Run the main script:
   ```bash
   python main.py
   ```

You should see output demonstrating the functionality of each module.

Feel free to import individual modules in your own scripts or extend them further.
